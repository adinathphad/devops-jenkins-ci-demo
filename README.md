# Jenkins CI Pipeline for Dockerized Flask Application

This project demonstrates a simple Continuous Integration (CI) pipeline using Jenkins to build and deploy a containerized Python application.

## Project Overview

The pipeline automatically:

1. Pulls the source code from GitHub
2. Builds a Docker image
3. Runs the containerized application

This demonstrates basic DevOps practices such as CI automation, containerization, and pipeline scripting.

## Tech Stack

- Jenkins
- Docker
- Python (Flask)
- GitHub
- Linux

## CI Pipeline Workflow


GitHub Repository -> Jenkins -> Build Docker Image -> Run Docker Container
